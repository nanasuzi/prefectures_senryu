#prefectures_senryu

## サイト概要
都道府県のオススメスポットを川柳で伝えるサイト。
ぱっとみて共有しあえるサイト。

### サイトテーマ
使用した便利アイテムや、行ったことのある場所のレビューを川柳で表現するサイト

### テーマを選んだ理由
ぱっと見て、「共感・面白い・行ってみたい・食べてみたい」を感じてもらえるサイトを作りたかったから

### ターゲットユーザ
ひまつぶしをする一般ユーザ

### 主な利用シーン
情報の記録、共有、暇つぶし（移動中などの隙間時間）ができる
